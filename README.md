# react-card

A react-component that renders a generic card, with both a front and back so
that the card can be flipped/rotated with CSS.

# Usage

```jsx
React.render(
  <Card>
    <Card.Front>

    </Card.Front>
    <Card.Back>

    </Card.Back>
  </Card>,
  document.body
);
```
